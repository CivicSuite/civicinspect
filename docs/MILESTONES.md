# CivicInspect Milestones

## Milestone 0 - Reconciliation

Align repo docs and gates to the CivicSuite unified spec.

## Milestone 1 - Runtime Foundation

FastAPI package, health/root endpoints, CivicCore pin, docs, tests, and release gate.

## Milestone 2 - Repeat-Case Lookup

Deterministic sample repeat-case context by property reference and violation type.

## Milestone 3 - Report Drafting

Inspector-owned report draft helper that preserves notes and requires review.

## Milestone 4 - Notice Draft Support

Draft notice helper with explicit staff-review actions and no issuance behavior.

## Milestone 5 - Records Export

Records-ready export checklist for municipal inspection case files.

## Milestone 6 - Public UI

Accessible public sample page at `/civicinspect`.

## Milestone 7 - Release

Release artifacts, checksums, CI green, and umbrella compatibility update.
